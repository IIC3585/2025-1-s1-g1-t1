insertColumn(file, n, column) → {Array.<Array.<any>>}

Description:

  • This function inserts a column into a file at the specified position. It can insert a column at the head (0), tail (-1), or any other position (n). The column must match the number of rows in the file.

Source:
  • functions.js, line 56
Examples
const file = [
  ["a", "b", "c"],
  ["d", "e", "f"],
  ["g", "h", "i"]
];
const column = ["1", "2", "3"];
const n = 1;
const newFile = insertColumn(file, n, column);
// console.log(newFile);
// Output:
// [
//   ["a", "1", "b", "c"],
//   ["d", "2", "e", "f"],
//   ["g", "3", "h", "i"]
// ];
const column = ["1", "2", "3"];
const n = -1;
const newFile = insertColumn(file, n, column);
// console.log(newFile);
// Output:
// [
//   ["a", "b", "c", "1"],
//   ["d", "e", "f", "2"],
//   ["g", "h", "i", "3"]
// ];

Edge Cases:
- If the column length does not match the number of rows in the file, an error will be thrown.
